# Ai-enemy-in-2d-Platform---Rhombix-Technologies
🎮 Task 2 — Enemy AI in a 2D Platformer

Built a simple AI system for a 2D platformer enemy in Unity, focused on two core behavior states: Patrol and Chase.

🔹 Enemy patrols between two fixed points using Vector2 direction calculations and Rigidbody2D velocity
🔹 Switches to chasing the player when within a defined detection range, using distance checks
🔹 Returns to patrolling automatically once the player moves out of range
🔹 Learned how Unity's physics collisions between Rigidbody2D objects can unintentionally push characters — and how to control that with mass and speed tuning

This task pushed me to actually understand state-based logic instead of just hardcoding movement. Still learning, still building — one task at a time.
